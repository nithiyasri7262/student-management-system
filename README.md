# Student Management System Using Python

## Overview

This project is a Python-based Student Management System developed as a mini project for B.Tech CSE. The application helps manage student records using CRUD operations.

## Features

* Add Student Records
* View Student Records
* Search Student Records
* Update Student Records
* Delete Student Records
* Store Data Using JSON File

## Technologies Used

* Python 3.13
* JSON File Handling
* IDLE Python Editor

## Project Structure

* student_management.py
* students.json

## CRUD Operations

### Create

Add new student records.

### Read

View existing student records.

### Update

Modify student details.

### Delete

Remove student records.

## How to Run

1. Open IDLE Python.
2. Open the file student_management.py.
3. Press F5 or select Run → Run Module.
4. Choose options from the menu.
